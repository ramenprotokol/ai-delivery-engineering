# Documentation Index

This directory holds the methodology, principles, and reference material for AI Delivery Engineering. The approach centers on one idea: AI accelerates the work; humans own the decisions, validation, and accountability. Every document here reflects that contract — what AI is used for, where human judgment is required, and how to tell the difference.
